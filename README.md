# Certificacion de Cableado Estructurado
![image alt](https://github.com/TuAmigoPapilla/Certificaci-n-de-Cableado-Estructurado/blob/0fe18773bb131a4bd9a5ff180ebd640cc8e0f964/ortronics-estructurado_cjplmr.png)


Descripción general 

En esta actividad, los estudiantes certificaron los puntos de red UTP con una certificadora. El objetivo era comprobar que estos puntos de red cumplen con los estándares TIA/EIA-568.

Objetivos

Ejecutar pruebas de certificación en puntos de red físicos para asegurarnos de que funcionen correctamente.

Verificar que todo cumpla con los estándares TIA/EIA-568, que son importantes para garantizar la calidad de nuestra red.

Documentar los resultados de estas pruebas utilizando Git y Markdown, para que podamos seguir el progreso y revisar los resultados de manera clara y organizada.

Contenido

parametros.md: Esto es donde se explican las definiciones técnicas.

equipo.md: Aquí se describen las funciones de la certificadora.

pruebas.md: En este archivo se presentan los resultados y el análisis de las pruebas.
